# Скрипт разбора json - файла conversion.json (импорт из DeepSeek)

## Назначение

- работать с беседами офф-лайн,
- удалять чаты,
- выполнять поиск,
- классифицировать беседы,
- сохранять беседы.

## Использование

1. Выполнить импорт в интерфейсе [веб-страницы deepseek](https://chat.deepseek.com/).
2. Положить файл в в один каталог с файлом программы parser.py.
3. Запустить файл.
4. В каталоге out (его нужно создать заранее) собрать набор бесед в виде текстовых файлов с названиями, составленными из имени беседы и даты беседы.
